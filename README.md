# Portfolio Website

## Overview
This project is a multi-page portfolio website built with a classic frontend structure using HTML, CSS, JavaScript, and PHP for contact-related functionality.

It is designed to present services, portfolio content, project details, reviews, FAQs, and contact information through a structured and user-friendly website experience.

## Project Goal
The goal of this project is to build a complete portfolio-style website with clear page organization, strong visual structure, and practical frontend implementation.

## My Role
I worked on building and organizing the frontend structure of the website, focusing on:
- layout and page structure
- styling and visual presentation
- organizing assets and website sections
- building a multi-page user experience
- supporting contact-related functionality through PHP files

## Tech Stack
- HTML
- CSS
- JavaScript
- PHP
